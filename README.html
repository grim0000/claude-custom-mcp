üìù Obsidian Setup Guide (Critical!)

To use the Obsidian tools, you must have the Local REST API plugin installed.

  1. Install the Plugin:
    • Open Obsidian > Settings > Community Plugins > Browse.
    • Search for "Local REST API" (bycodersnet).
    • Install and Enable it.
  2. Get your API Key:
    • Go to the plugin settings (Local REST API).
    • Find the API Key section.
    • Click "Copy" or "Regenerate".
  3. Configure Environment:
    • The installer asked for this key. If you provided it, you are done!
    • If you skipped it, you need to set the OBSIDIAN_API_KEY environment variable in your system manually or reinstall.
